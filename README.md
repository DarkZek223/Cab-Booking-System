🚖 Cab Booking System

A Java-based Cab Booking System developed using Java Swing, JDBC, MySQL, and Apache Ant (XML). The application provides an intuitive desktop interface for managing customers, booking local and intercity cabs, handling trucking services, processing payments, and maintaining booking records.

This project demonstrates desktop application development with database connectivity and CRUD (Create, Read, Update, Delete) operations in Java.

🚀 How It Works
User logs into the application.
Register or manage customer details.
Book local or intercity cabs.
Add trucking service bookings if required.
Store booking details in the MySQL database.
Process customer payments.
View and manage booking history.
